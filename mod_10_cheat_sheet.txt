Prompt: Given the following XQuery code:

let $x := doc("courses.xml")/Courses
for $in in $x/Course
where $in/Level > 200
order by $in/Code
return <Course> {$in/Name, $in/Level, $in/Code} </Course>

Which line of the code defines the filter function?
Options:
  a. order by $in/Code
  b. return <Course> {$in/Name, $in/Level, $in/Code} </Course>
  c. for $in in $x/Course
  d. where $in/Level > 200
Answer: where $in/Level > 200 (d)

Prompt: What are the key features of a native XML database? Select all that apply.
Options:
  a. Store XML data as is
  b. No need to load the entire XML into memory
  c. Allow queries for a subset of data in memory
  d. Use the relational database query language
Answer: Store XML data as is, Allow queries for a subset of data in memory (a, c)
---
Prompt: Using the object-oriented concept of inheritance, what language is the base (parent) language of XQuery?
Options:
  a. XML Schema
  b. XPath
  c. XSL
  d. XML
Answer: XPath (b)
---
Prompt: What are the main problems of using relational databases for XML databases where the XML has an irregular structure? Select all that apply.
Options:
  a. The database cannot be indexed
  b. XSL transformations cannot be performed
  c. It may create many null columns in a table for representing the data
  d. It may create many tables for representing the data
Answer: It may create many null columns in a table for representing the data, It may create many tables for representing the data (c, d)
